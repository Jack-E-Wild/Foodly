todos:

Verify that you have a JDK installed:
1. open a terminal and run "java -version"
2. if it returns "java version "21.0.11" you are done
3. if it returns a different version -or- command not found:
	- install Java SDK 21:
	- https://www.oracle.com/java/technologies/downloads/#java21


install Springboot demo:
	- pick Maven and Java
	- http://start.spring.io/
	- using version 4.0.6 as the newest stable build
	- leave demo project names
	- add dependencies: Web Spring

	- click GENERATE
	- copy the zip content 'demo' folder to the appropriate project folder (e.g. C:\Users\{YOUR_USERNAME}\IdeaProjects\ )

	you can now test the spring boot demo folder.
